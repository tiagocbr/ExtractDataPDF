# Frontend - Extração de PDFs

Esta é a interface web para a **extração estruturada de PDFs** usando Next.js e React. Permite enviar documentos individualmente ou em lote, interagindo com o backend que processa os PDFs e retorna os dados extraídos.

Esta aplicação foi criada com [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

---

## Pré-requisitos

- Node.js >= 18
- npm / yarn / pnpm / bun
- Backend rodando localmente ou acessível via URL

---

## Instalação

Clone o repositório, navegue até a pasta front-end e instale as dependências:

```bash
npm install
# ou
yarn install
# ou
pnpm install
# ou
bun install
```

Para iniciar o servidor de desenvolvimento:

```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
# ou
bun dev
```

Abra http://localhost:3000
 no navegador para acessar a interface.

## Tecnologias Utilizadas

- **Next.js**  
  Framework React para SSR/SSG e otimizações automáticas.

- **React**  
  Biblioteca de UI para construção de interfaces reativas.

- **MUI (Material-UI)**  
  Componentes de interface modernos e responsivos.
